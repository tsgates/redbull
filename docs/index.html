Why Redbull

Small. Native. Effortless.

Everything you need to stop your Mac from sleeping — and nothing you don't.

🎚️

One slider

Drag from Off to ∞. The whole control is a single tick slider — no menus to dig through.

⏱️

Sensible presets

15 minutes, 1 / 2 / 3 / 6 / 12 hours, or indefinitely. Snaps right to the stop you want.

⚡

Live countdown

The menu-bar bolt shows the time remaining at a glance, with a fixed width so it never jitters.

🪶

Featherweight

A ~100 KB native binary in Rust + AppKit. ~0% CPU when idle. No Electron, no daemon.

🔌

Zero config

No account, no login item, no preferences. Launch it and it lives quietly in your menu bar.

🔓

Open source

Built on the system caffeinate tool. Read every line — nothing hidden.

Just caffeinate, made pleasant.

macOS already ships a power-assertion tool. Redbull is a clean front-end for it — pick a duration and it runs:

Display + idle sleep are held for the chosen time, then released automatically.

# keep display + system awake for 2 hours
$ caffeinate -d -i -t 7200
Get started

Up and running in a minute

Apple Silicon or Intel Mac, macOS 10.13+. Two ways to install.

Download

Grab the .dmg for your chip (Apple Silicon or Intel) from the releases page.

Drag to Applications

Open the disk image and drop Redbull into your Applications folder.

First launch

Right-click → Open once (it's ad-hoc signed), then click the ⚡ in your menu bar.

Prefer to build it yourself?

$ git clone https://github.com/tsgates/redbull
$ cd redbull && ./release.sh
